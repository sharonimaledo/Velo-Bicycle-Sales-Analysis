# Velo Bicycle Sales Performance and Seasonality Analysis

## Project Overview
This project presents an exploratory analysis of multi-year bicycle sales data completed as part of a Business Intelligence Analyst technical assessment. The objective is to evaluate sales performance across countries, products, and time, identify patterns in revenue, volume, and seasonality, and assess the impact of the 2008 Lehman Brothers financial crisis on sales behavior.
All analysis was conducted in Power BI, with Power Query used for data cleaning and transformation, followed by interactive dashboard development to support insight generation and decision-making.

## Tools and Technologies
- Power BI
  - Data modeling and DAX calculations
  - Interactive dashboards and time-series analysis
- Power Query
  - Data cleaning and transformation

- Some products exhibit weak or absent seasonality, contrary to expected bicycle purchasing patterns.
